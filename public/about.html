About Fighter Metrics

The Problem

When I was 11 years old, I watched UFC 194 with my dad — Conor McGregor knocked out Jose Aldo in 13 seconds. Since then I've been hooked on the sport. But as a fan, I've always found it frustrating that tracking fighter stats, championship history, win methods, and divisional trends means bouncing between half a dozen different websites. Fighter Metrics solves that by pulling it all into one place.

What It Does

  • View upcoming UFC events and current fight cards live from ESPN
  • Look up any UFC fighter by their ESPN athlete ID
  • Save fighters to your personal dashboard via Supabase
  • Visualize win method breakdowns with Chart.js
  • Browse fighters on the current card

Data Sources

  • ESPN Unofficial UFC API — Live event data, fighter profiles, fight cards. No auth required.
  • Supabase — PostgreSQL database storing your saved fighters.

Tech Stack

  • Frontend: HTML, CSS, Vanilla JS, Chart.js, CountUp.js
  • Backend: Node.js + Express
  • Database: Supabase (PostgreSQL)
  • Deployment: Vercel

Who Is This For?

UFC fans who want stats without the hassle. Sports journalists looking for quick reference data. Analysts who want to spot trends in fight methods and divisional performance. And anyone who, like me, got hooked watching two fighters leave everything in the octagon.
